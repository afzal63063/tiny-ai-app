## 📌 Overview  
This project is an AI-powered Text Summarizer that converts long text into short, meaningful summaries using NLP techniques.  
It helps users quickly understand large content like articles, blogs, and reports.

---

## ✨ Features  
- 🔍 Automatic text summarization  
- ⚡ Fast and efficient processing  
- 🧠 Powered by NLP models  
- 🌐 Simple and interactive UI
